# simple nodejs setup with mocha tests

[![CircleCI](https://circleci.com/gh/jrollin/node-mocha.svg?style=svg)](https://circleci.com/gh/jrollin/node-mocha)

## Dependencies 

* mocha / chai for testing
* nyc for code coverage


## Installation

```bash
npm install
```

## Run tests 

with watch

```bash
npm run test:watch
```
